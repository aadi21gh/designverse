# DesignVerse

DesignVerse is a C2B2C customization platform where users can create personalized artwork, clothing, and accessories, purchase them, share their designs with the community, and earn rewards when others buy their creations.

## Tech Stack

- React + Vite
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Features

- User Authentication
- Product Customization
- Design Canvas
- Explore Feed
- Creator Rewards
- Wallet
- Admin Dashboard
